# نظام حضور مشاريع ALYAMAMA

نسخة محلية مستقلة لتجربة نظام حضور وانصراف المهندسين والمراقبين في:

- مشروع الصواري
- مشروع الياقوت
- المدينة: جدة
- المنطقة: شمال جدة

## التشغيل

من داخل هذا المجلد شغل:

```powershell
python -m http.server 8765
```

ثم افتح:

```text
http://127.0.0.1:8765/alyamama/
```

## دخول الإدارة

- اختر تبويب `الإدارة`
- رمز الإدارة الافتراضي: `1234`

من لوحة الإدارة يمكن تعديل:

- موقع كل مشروع
- إحداثيات المشروع
- نصف قطر السماح بالمتر
- وقت بداية الدوام
- سجلات الحضور والانصراف
- الموظفين المسجلين ذاتياً

## دخول الموظف

الموظف يستطيع إنشاء حساب من تبويب `تسجيل موظف جديد` عن طريق:

- الاسم الكامل
- رقم العمل
- اختيار المشروع أو المشاريع

بعد الدخول:

1. يختار المشروع.
2. يضغط `تحديد موقعي وإتمام الحضور`.
3. يسمح للمتصفح باستخدام GPS.
4. يؤكد الموقع.
5. يتم قبول أو رفض الحضور حسب قربه من موقع المشروع.

## ملاحظة مهمة عن GPS

المتصفحات الحديثة تشترط غالباً أن يعمل تحديد الموقع عبر:

- `localhost`
- أو موقع منشور عبر `HTTPS`

لذلك تجربة GPS على الجوال تحتاج لاحقاً نشر التطبيق على رابط آمن `HTTPS`، أو تحويله لتطبيق جوال فعلي.

## تحويله إلى تطبيق جوال

تم تجهيز المشروع كـ `PWA`، وهذا يعني أنه يمكن تثبيته على شاشة الجوال من المتصفح بعد نشره على رابط `HTTPS`.

### Android

1. افتح رابط التطبيق في Chrome.
2. افتح قائمة المتصفح.
3. اختر `Add to Home screen` أو `Install app`.
4. سيظهر التطبيق على شاشة الجوال باسم `حضور ALYAMAMA`.

### iPhone

1. افتح رابط التطبيق في Safari.
2. اضغط زر المشاركة.
3. اختر `Add to Home Screen`.
4. سيظهر التطبيق على الشاشة كتطبيق مستقل.

### APK / تطبيق متجر

إذا كان المطلوب ملف `APK` أو تطبيق ينشر على المتجر، فالخطوة التالية هي تغليف نفس التطبيق باستخدام أداة مثل Capacitor أو Cordova، ثم ربطه بقاعدة بيانات وسيرفر حقيقي. هذه الخطوة تحتاج بيئة Android Studio أو خدمة بناء تطبيقات.
